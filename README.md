# 大同镇西乡茶园 · 扫码导览网站

两种查看景点介绍的方式：

1. **点击地图热点**：打开网页即见导览图，图上每个景点有一个圆形脉冲标记，点击即弹出该景点的介绍和图片。
2. **扫码**：在景点处扫描二维码 → 直接打开对应景点弹窗 → 关闭后看到整张导览图。

两种方式弹窗一致；关闭弹窗后都回到全屏**景区导览图** `map.jpg`。

纯静态站点，无构建步骤、无框架，可托管在任意静态服务器（GitHub Pages、对象存储、Nginx 等）。

## 目录结构

```
.
├── index.html              主页面：展示导览图 + 景点弹窗
├── map.jpg                 景区导览图（全屏背景）
├── css/style.css           样式
├── js/
│   ├── spots.js            景点数据（名称 / 介绍 / 图片 / 地图坐标）★需替换真实内容
│   └── app.js              逻辑：渲染地图热点、解析 ?spot= 参数、弹窗
├── assets/placeholders/    占位图片（SVG）★需替换为真实照片
├── editor/index.html       热点编辑器：在地图上可视化定位每个景点
├── qr/
│   ├── index.html          二维码生成 / 打印工具
│   └── vendor/qrcode.js    本地内置二维码库（离线可用）
└── claude_tasks/           计划与记录
```

## 工作原理

- 打开 `index.html` 即展示导览图，图上按 `spots.js` 里每个景点的 `coords` 渲染圆形热点；点击热点弹出该景点弹窗。
- 每个景点同时对应一个网址：`index.html?spot=<景点id>`，扫码进入会自动弹出对应弹窗。
- 点击「关闭并查看导览图」/ 关闭按钮 / 弹窗外区域 / Esc → 关闭弹窗，露出全屏 `map.jpg`。
- 扫码关闭后地址栏会自动去掉 `?spot=` 参数，刷新即直接看地图。

## 调整地图上景点的位置（热点编辑器）

景点坐标当前是读图估算的近似值，用编辑器可视化校准：

1. 打开 `editor/index.html`。
2. 左侧选中一个景点 → 在地图上点击它的真实位置（或直接拖动地图上已有的圆点微调）。
3. 全部调好后点 **下载 spots.js**，用下载的文件覆盖项目里的 `js/spots.js` 即可。

> 编辑器只改坐标，不影响景点的名称 / 介绍 / 图片。

## 如何替换为真实内容

编辑 [`js/spots.js`](js/spots.js)，每个景点对象：

```js
{
  id: "foshou-hu",          // ★二维码已绑定此 id，不要改动
  name: "佛手湖",
  intro: ["第一段介绍……", "第二段介绍……"],   // 每个数组元素是一段
  images: ["assets/photos/foshou-hu-1.jpg"]   // 换成真实照片路径
}
```

- **图片**：把真实照片放入 `assets/`（建议新建 `assets/photos/`），再更新 `images` 路径即可。
- **景点增删**：直接在数组中增删对象；新增请同时给一个唯一 `id`。

## 生成二维码

1. 把整个站点部署到线上，拿到访问地址（例如 `https://your-domain.com/`）。
2. 打开 `qr/index.html`，在「网站基础地址」填入该地址，点击**生成二维码**。
3. 点击**打印 / 导出 PDF**，把对应景点的二维码打印张贴到景点现场。

> 二维码内容为 `<基础地址>/index.html?spot=<景点id>`，库为本地内置，离线也能生成。

## 本地预览

```bash
# 在项目根目录启动一个静态服务器
python3 -m http.server 8000
# 浏览器访问：
#   导览图（点热点）： http://localhost:8000/
#   某景点弹窗：       http://localhost:8000/index.html?spot=foshou-hu
#   热点编辑器：       http://localhost:8000/editor/index.html
#   二维码工具：       http://localhost:8000/qr/index.html
```
